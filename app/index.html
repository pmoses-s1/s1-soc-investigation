Runs a standard SOC investigation query catalog across long (90+ day) lookbacks over the SentinelOne Singularity Data Lake, slice by slice. Reliable execution with automatic retries, rate control, and caching; a per-run verification report that proves nothing was skipped; and an exported workbook plus downloadable log evidence. No UI timeouts, no silently-dropped queries, no MCP. Pick a catalog, an entity, and a lookback, start it, and come back to reviewed findings.
